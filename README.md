#Hearthnode
Web Based Hearthstone deck builder.

### Purpose
This app gives users the ability to view, create, manage, and post Hearthstone decks crafted either by them or other users.

### Technologies Used
- Javascript (server side & front end)
- Express
- Jade
- bcrypt
- node.js
- mongodb
- monk
- Hearthstone API.
- Unirest
